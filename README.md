# Production Deployment Guide

Тёмный сайт-документация для курса **Production Deployment**. Интерфейс построен как техническая документация: левая навигация, основной документ, правое содержание, code block, callout, timeline, checklist, таблица и SVG-схема.

## Запуск локально

```bash
python3 -m http.server 4173
```

Откройте [http://127.0.0.1:4173](http://127.0.0.1:4173).
