Для кого

Не Junior-трек, а путь инженера эксплуатации

План рассчитан на разработчика, который уже пишет на Python, знаком с Docker, Nginx, Linux и PostgreSQL. Цель — научиться не только разрабатывать API, но и проектировать, разворачивать и сопровождать сервисы в production.

Roadmap

Этапы обучения

Этап 2 · 3–4 недели

Production Deployment

Путь от локального FastAPI-проекта до production: Docker, VPS, HTTPS, systemd, Nginx и PostgreSQL.

  • Linux: filesystem, users, permissions, processes, network
  • Systemd-сервис и journalctl
  • Docker Compose: FastAPI, PostgreSQL, Redis
  • Nginx: TLS, headers, rate limiting, gzip, caching
  • PostgreSQL: backup, indexes, EXPLAIN ANALYZE
Этап 3 · 2–3 недели

CI/CD

Автоматизируйте путь от push до деплоя на VPS без простоя.

  • GitHub Actions: lint, tests, build, docker push
  • Secrets, artifacts, reusable workflows
  • Blue-green, rolling update, canary, rollback
Этап 4 · 3 недели

Observability

Уровень Senior начинается там, где сервис становится измеримым и диагностируемым.

  • JSON logs, correlation IDs, structlog/loguru
  • Prometheus: Counter, Gauge, Histogram, Summary
  • Grafana dashboards: API, DB, Redis, server
  • OpenTelemetry + Jaeger tracing
Этап 5 · 4–5 недель

Cloud Engineering

Перенесите проект с VPS в AWS и разберите базовые cloud-компоненты.

  • EC2, Auto Scaling
  • S3
  • VPC, Subnet, Route Tables
  • IAM, Security Groups
Этап 6 · 6–8 недель

Kubernetes

После VPS, Docker и CI/CD переходите к оркестрации и GitOps.

  • Pod, Deployment, Service, ConfigMap, Secret, Ingress
  • Helm chart
  • HPA, limits, probes, affinity, taints, tolerations
  • ArgoCD или FluxCD
Этап 7 · 6–12 месяцев параллельно

Архитектура

Переход к Senior/Lead: выбор паттернов, надежность, данные и event-driven подход.

  • Monolith, modular monolith, microservices
  • RabbitMQ, Kafka
  • PostgreSQL, Redis, Elasticsearch
  • Circuit breaker, retry, rate limiting, bulkhead, CAP

Финальный проект

Production-система с полным контуром эксплуатации

Соберите SaaS, который проходит весь путь: Cloudflare → Nginx → FastAPI → Redis → PostgreSQL, а также покрывается Prometheus, Grafana, Loki, Jaeger, GitHub Actions, Docker Registry и Kubernetes.
